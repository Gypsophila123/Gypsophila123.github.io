开始使用hexo

发表于 2020-10-22 分类于 hexo Valine： 1.4k 1 分钟
本文介绍了如何下载hexo和使用hexo快速搭建一个博客

Hexo 是一个快速、简洁且高效的博客框架。在几秒内，即可利用靓丽的主题生成静态网页。

即部署出来的网站是静态的，可以在如github page，gitee page上使用，无需自己使用服务器。

前提

仅仅需要两个程序员必备工具：

Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
Git

下载hexo

配置好上述软件之后，使用npm命令

可以使用” hexo server “（简写hexo s）命令开启本地服务器，再使用浏览器打开http://localhost:4000/，就可以看到刚才建立的网站模板

上传到网络

只能在本地浏览当然不是一个博客该做的，所以我们需要将网站发布到网络。

建立一个git仓库

下面将会讲解如何通过GitHub建立这个仓库，当然你也可以通过别的网站来建立，具体步骤大同小异。

  1. 注册github账号（有账号直接忽略）

  2. 新建仓库，仓库名为 你的github名.github.io,如我的：Gypsophila123.github.io，直接创建就可

  3. 配置ssh秘钥，不会可以网上搜索 github ssh 配置

  4. 修改配置文件_config.yml文件最下面，以我的为例

  5. 命令行输入hexo g -d 就会自动生成静态文件并上传，等一会就可以在和仓库名相同的链接上看到博客了（如果是gitee可能需要在仓库界面>服务>更新才能看到，建议勾选https）
