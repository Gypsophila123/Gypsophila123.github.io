实现矩阵旋转的两种方法：自外向内 & 两次旋转

肖邦鹏发布于 2019-05-0823.0k精选脑筋急转弯数组Java

方法 1：自外向内顺时针循环

自外向内一共有不超过 n/2n/2 层（单个中心元素不算一层）矩形框。对于第 timestimes 层矩形框，其框边长 len=nums-(times2)len*=*nums−(times∗2)，将其顺时针分为 44 份 len-1le*n−1 的边，对四条边进行元素的循环交换即可。

  • Java

方法 2：两次翻转

先沿右上 - 左下的对角线翻转（270° +270°+ 一次镜像），再沿水平中线上下翻转（-180° +−180°+ 一次镜像），可以实现顺时针 9090 度的旋转效果

  • Java
