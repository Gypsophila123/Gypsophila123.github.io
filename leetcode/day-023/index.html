题目

单词规律

难度简单

给定一种规律 pattern 和一个字符串 str ，判断 str 是否遵循相同的规律。

这里的 遵循 指完全匹配，例如， pattern 里的每个字母和字符串 str 中的每个非空单词之间存在着双向连接的对应规律。

我的思路

这道题虽然是简单题，但是给出的条件比较模糊，如：pattern的长度是否有限制，pattern对应的 str 能否一对多，str 的长度限制。那么我们先自己给自己限制一下，从题目可以看出，pattern 4 个字母对应 str 四个单词，就姑且认为这是一对一的关系，pattern我们则认为是一个正整数。

pattern中字母相等，单词就会相等，根据经验，可以用到桶。按照示例 1 走一遍：

输入: pattern = "abba", str = "dog cat cat dog"

  • 先看pattern，pattern第一个是 a，没有这个桶，拿一个空桶丢进去。
  1. 第二个是 b ，没桶，拿一个空桶丢进去。
  2. 第三个 b， 有桶，丢进桶
  3. 第四个 a，有桶， 丢进桶
  • 我们现在有两个桶，一个装着 1、4，一个装着2、3

所以单词 1 等于单词 4，单词 2 等于单词 3
这样一来遍历一次就知道有没有问题了
值得注意的是：从示例 3 看出，一个字母只对应一个单词。

  • 停停，我在写代码的时候发现

既然一个字母只对应一个单词，可以直接转化单词为字母，一发现不对应就跳出
